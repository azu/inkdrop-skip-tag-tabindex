# inkdrop-skip-tag-tabindex

Inkdrop tweak: skip tag's tab index.

![screenshot](https://monosnap.com/image/fMtCVjvk7GNroCefRG6UksiYahbFvY)

## Install

Install with [npm](https://www.npmjs.com/):

    ipm install inkdrop-skip-tag-tabindex

## Usage

1. Input title 
2. <kbd>Tab</kbd> key
3. Input Body
